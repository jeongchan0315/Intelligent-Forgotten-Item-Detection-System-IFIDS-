# Intelligent-Forgotten-Item-Detection-System-IFIDS-
IFIDS (Intelligent Forgotten Item Detection System) is a departure detection system that helps users reduce instances of leaving essential items behind due to forgetfulness. Primarily based on an Arduino Uno, RFID tags, and HuskyLens computer vision, the system will detect whether the user has forgotten an item and alert them. The final product will also be able to determine the time of day it is to see if a certain item is necessary. The goal of the IFIDS is to create an affordable solution to a small but meaningful problem in everyday life.
